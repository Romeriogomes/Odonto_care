Aplicativo Odonto Care

Módulos

  1. Clientes

    2. No módulo Clientes fica salvo todo o histórico do paciente referente a toda movimentação efetuada na clínica.

  3. Agenda

    4. O módulo Agenda controla as datas e horários dos atendimentos, sendo possível customizar o tempo médio de cada atendimento. Por padrão, vai com tempo de 30 minutos, está habilitado o barramento de agendas duplicadas. Tem a possibilidade de abrir o WhatsApp com mensagem pré-configurada para envio.

  5. Prontuário

    6. Neste módulo é possível inserir e atualizar o histórico médico do paciente.

  7. Anamnese

    8. O módulo Anamnese contempla um questionário por paciente.

  9. Orçamentos

    10. É possível fazer os orçamentos e controlar e gerir os recebimentos por este módulo. Por padrão, a data de validade é 30 dias após esse período o orçamento é mudado para status cancelado automaticamente. É possível customizar a data de validade.

  11. Relatórios

    12. Neste módulo é possível filtrar o período de receita gerada pela clínica, nele é rastreado todos os orçamentos com status pago.

  13. Receituário

    14. Disponibiliza a possibilidade de fazer o receituário seja ele simples ou especial.

  15. Recibos

    16. Tem a possibilidade emitir recibos simples ou completo já com período fiscal filtrado, nos recibos completos ele filtra todos os orçamentos com status pago dentro do período informando para emissão, já no simples informa apenas o valor total desejado sem vínculo com os orçamentos.
